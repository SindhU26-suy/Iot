Install Arduino and Raspberry Pi(WEEK - 1)

Arduino is a platform that makes it easy for you to build projects using electronics.
 IoT is a way of using electronics - to make electronic modules talk to each other remotely
and wirelessly (often using a Cloud) to solve problems.
 Now, Arduino can also help you easily build IoT projects in two ways: Using traditional
Arduino boards and attaching communication breakout modules (like nRF Bluetooth,
WiFi, LoRA, GSM, etc) to them.
 Arduino is a micro controller that can be connected to one or more sensors and help you
capture the data or information and then pass it on to processor. If you know the full stack
of IoT then you should also look at Raspberry.
 RaspPi is a microprocessor so the basic difference between Arduino and RasPi is that
RaspPi is controller plus processor and Arduino is just a micro controller.
